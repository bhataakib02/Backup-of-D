## 🚀 **DEPLOYMENT INSTRUCTIONS**

### **Quick Start (Windows)**
```batch
# Run the deployment script
deploy.bat

# Or manually:
# 1. Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py

# 2. Frontend (new terminal)
cd frontend
npm install --legacy-peer-deps
npm start
```

### **Quick Start (Unix/Linux/Mac)**
```bash
# Run the deployment script
./deploy.sh

# Or manually:
# 1. Backend
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

# 2. Frontend (new terminal)
cd frontend
npm install --legacy-peer-deps
npm start
```
